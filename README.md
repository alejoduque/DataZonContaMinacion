# Taller de DataZonificacion
## Sonificacion de Datos de Contaminacion

Este taller tiene como objetivo crear un dispositivo que nos permita 'escuchar la contaminacion del aire' al realizar una deriva por las calles de la ciudad (Soundwalk). <br>

El taller servira para aprender a trabajar con el software SuperCollider. Los y las asistentes podran instalarlo en sus propios equipos y entre todos compartiremos como usarlo para, a partir de datos que provienen de la medicion de sensores de material particulado (pm2.5), podran generar sintesis sonara en tiempo real. El soporte sera la plataforma de computacion RaspberryPi.

Tomamos en cuenta las siguientes consideraciones tecnicas:
-Enviar datos via OSC (UDP) del AQA al Rpi et al.<br>
